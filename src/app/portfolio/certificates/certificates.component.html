Advanced PHP with Laravel

Institute: IT Training Nepal Pvt Ltd

Duration: 2.5 Months (Jan 2018 to March 2018)

Introduction to composer, mvc pattern:
Routing api and web (GET,POST,PUT,PATCH,DELETE etc) (csrf protection). Artisan (Laravel build in CLI, create your own CLI application). Create Controller for business logic. Create model to work with DB. View the presentation layer. Dependency injection.

Concept of RDBMS:
DB architect with Laravel migration. Data seeding with Laravel Seeders. Eloquent ORM to work with mysql. DB Facade to work with mysql. Db table relationships (one to one, one to many, many to many).

Templating with blade syntax. Cookies and Session Facade/Helper functions. Request and Response lifecycle. Laravel Authentication (concept of guards). Middleware to check a valid request. Service container / Service Providers and Facade. Mailing and Events. Laravel frontend support(ES6/SCSS bundling with webpack). Deploying Laravel application.

MEAN Stack

Institute: Broadway Infosys Pvt Ltd

Duration: 2 Months (April 2019 to May 2018)

Understanding of complex web applications development using a combination of Javascript technologies, MongoDB (M), Express JS (E), Angular JS (A), Node JS (N)

Introduction to development tools:
Text Editor, Command Prompt (command line), Git, Git workflow (branching,push,pull), NPM,Yarn, Postman.

Introduction to nodejs:
Node package manager (npm) Asynchronous and event loop in node js, Events and handling Events, Node CLI, Node modules, File handling with node js, Creating http server with nodejs, Introduction to websockets, Create realtime socket communication.

The model-view-controller pattern in ExpressJS:
Routing, HTTP interaction, Request and Response, Handling form data, Handling query parameter, Status code response, Middlewares in Express,

Understanding NoSQL (MongoDB):
Database modeling and Schema design (using mongoose), Update, insert delete and upsert documents

SPA & Angularjs 2+ framework architecture:
Typescript, Class based inheritance, Component based architecture, Module based architecture, Routing, Data binding (one way & Two-way binding), Pipes and Directives, Forms and Forms Validation, Observables & promises (rxjs map,subscribe,catch), Dependency injection, Services ($http), Decorators, Introduction to frontend build tools (angular CLI, Webpack)
